# UUID generator

A simple library that generates uuids.

Supported versions:

- [x] version 1
- [x] version 3
- [x] version 4
- [x] version 5

Supported variants:

- [x] DCE
- [x] Microsoft

The supported namespaces are the ones that are included on the [RFC 4122](https://datatracker.ietf.org/doc/html/rfc4122#appendix-C).
